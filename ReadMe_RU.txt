Проект: social networks in parliaments

Идея проекта по ислледованию социальных сетей в различных парламентах появилась благодаря видео на канале Utube:
https://www.youtube.com/c/PythonToday/

В данном видео был подробный расказ о том как собрать ссылки на соцсети всех членов парламента с сайта Бундестага.
В видео никто ничего не взламывал,
ссылки на соц сети - это открытая информация размещенная в профиле каждого члена парламента.
можете убедиться сами, вот сайт Парламента Германии: https://www.bundestag.de/en/members

И при просмотре видео у меня появились вопросы:
А какими социальными сетями пользуются члены парламентов разных стран?
Как это зависит от страны, партии, возраста?

Конечно меня интересует обезличенная информация и цель проекта не собрать ссылки на социальные сети всех парламентариев.
Меня интересует только информация о том какая социальная сеть и сколько раз используется различными группами, поэтому в коде
все ссылки на соц сети заменены на "1".

Конечная цель проекта - это интерактивный дашборд на платформе Microsoft Power BI,
на котором можно будет увидеть интересную аналитику и использовать это затем в исследовательских, журналистских или
 просто познавательных целях.

 Первым парламентом, естественно стал Парламент Германии.

 На данный момент я написал парсер, который собрал все необходимые мне данные в словарь JSON для последующей обработки
 в Power BI.
 Кстати там есть интересный момент по получению даты рождения. По-моему получилось неплохо.
